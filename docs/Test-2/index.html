Problem 1 (p. 204)

Suppose X^{↻\alpha} \xrightarrow{f} Y^{↻\beta} is a map in 𝑺↻. Show that if \alpha has a fixed point, then \beta must also have a fixed point.

Solution: Problem 1

The sequence of rewrites in the example below mirrors Danilo's solution on p. 205.

example (X Y : SetWithEndomap) (f : X ⟶ Y) : (∃ x : X.carrier, X.toEnd x = x) → (∃ y : Y.carrier, Y.toEnd y = y) := X:SetWithEndomapY:SetWithEndomapf:X ⟶ Y⊢ (∃ x, X.toEnd x = x) → ∃ y, Y.toEnd y = y X:SetWithEndomapY:SetWithEndomapf:X.carrier ⟶ Y.carrierhf_comm:f ⊚ X.toEnd = Y.toEnd ⊚ f⊢ (∃ x, X.toEnd x = x) → ∃ y, Y.toEnd y = y X:SetWithEndomapY:SetWithEndomapf:X.carrier ⟶ Y.carrierhf_comm:f ⊚ X.toEnd = Y.toEnd ⊚ fx:X.carrierhα:X.toEnd x = x⊢ ∃ y, Y.toEnd y = y X:SetWithEndomapY:SetWithEndomapf:X.carrier ⟶ Y.carrierhf_comm:f ⊚ X.toEnd = Y.toEnd ⊚ fx:X.carrierhα:X.toEnd x = x⊢ Y.toEnd (f x) = f x All goals completed! 🐙
