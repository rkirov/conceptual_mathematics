# Conceptual Mathematics — Exercises (Lean 4)

Lean 4 / mathlib exercises following Lawvere and Schanuel's _Conceptual Mathematics_
(2nd edition).

The exercise files live in [`ConceptualMathematics/Sorried`](ConceptualMathematics/Sorried).
Most definitions and proofs are stubbed out with `sorry` and are intended to be
filled in.

This is a trimmed-down fork of David Kinkead's
[Verso companion](https://github.com/davidkinkead/conceptual_mathematics); the Verso
documentation toolchain and the rendered HTML have been removed so the project builds
against mathlib alone.

## Building

```sh
lake build
```
